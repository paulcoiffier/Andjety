Introduction

Andjety permetter d'effectuer des requêtes SQL sur des bases de données Oracle & MySQL. Vous possédez un parc de bases de données, et vous souhaitez exécuter une requête sur l'ensemble de ces bases ? Andjety est là pour automatiser l'exécution de ces requêtes. Fournissez la liste de vos bases de données à Andjety et il effectuera l'exécution pour vous. Andjety vous permettra également d'obtenir un état des requêtes effectuées sous forme de fichier EXCEL. Andjety est donc un collecteur de données multi-bases !
