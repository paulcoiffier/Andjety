Create task

A task can start an inventory of databases previously created. To create a task, click that found in the toolbar. The task management panel will then appear:




Now click this button to open the editing tasks:





Then learn the required fields:

  • Result : Path to the folder where the file will be created with or the results of the query
  • Nameé : Task name to enable identification
  • SQL Query : SQL query to execute


Click on to continue creating the task.



Select the databases you want to include in your inventory, and click on "Finish" to complete your task creation.
