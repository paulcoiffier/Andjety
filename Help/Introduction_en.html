Introduction

Andjety permit to perform SQL queries on Oracle & MySQL databases.You have a fleet database, and you want to run a query on all of these bases? Andjety is to automate the execution of these queries. Provide a list of your databases and Andjety will run for you. Andjety will also allow you to obtain a state of queries in an EXCEL sheet. Andjety is a data cross-database!
